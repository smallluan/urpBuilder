BuilderNext 面向**可视化搭建**与**数据集成**：在同一工作区内管理页面、组件、常量、云函数与素材，并支持预览、版本与协作。

<DocsCallout title="MDX">正文支持嵌入 React 组件（本块即为示例）。</DocsCallout>

## 你能用平台完成的事

- **搭建应用与组件**：从列表进入编辑器，拖拽布局与业务组件，配置属性与数据源。
- **连接数据**：常量、API、云函数与素材库按团队/个人空间隔离，按需引用。
- **预览与发布链路**：内置预览引擎，便于在上线前核对交互与样式。

### 典型工作流（示例）

| 步骤 | 说明 |
| --- | --- |
| 1 | 在「构建应用」中创建或打开页面 |
| 2 | 在画布中组合 `Layout` / 业务组件并绑定数据 |
| 3 | 使用预览检查多端表现 |
| 4 | 与团队评审后走发布流程（以你方环境为准） |
